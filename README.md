# SAP Gig Finder

SAP Gig Finder is a multi-agent pipeline that automatically scouts, analyzes, validates, and formats freelance SAP consulting opportunities from sources like Upwork, job boards, and RSS feeds. It uses Claude as the AI backbone to rank and summarize gigs based on relevance, rate potential, and skill fit, then outputs a curated shortlist so SAP consultants can focus on applying rather than searching.
